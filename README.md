- محتوى Accordion قابل للطي/الفتح
- تقييمات الطلاب مع نجوم وشارة "مشترك موثّق"
- آراء صريحة ومتوازنة (إيجابية وسلبية)
- أزرار التنقل بين الدورات
- توصية ذكية مضمّنة في كل دورة

### 🎯 نظام التوصيات الذكي
- اختبار 4 أسئلة (وقت / مستوى / هدف / قطاع)
- نسبة توافق مخصصة لكل طالب
- توصية بالدورة الأنسب مع شرح السبب
- خيارات بديلة + روابط سريعة
- متاح من: Hero / جانب الدورات / قسم الشهادات

### 💬 ويدجت واتساب العائم
- أيقونة عائمة مع نبضة حيّة
- مؤشر "متصل الآن" الأخضر
- شعار الأكاديمية في رأس النافذة
- ردود سريعة (4 أنواع استفسارات)
- ردود تلقائية مع روابط وأزرار مخصصة
- أزرار: مشاركة الموقع / للأعلى / الدورات
- يفتح للاستشارة من داخل تفاصيل الدورة

### ⭐ قسم الشهادات المحسّن
- تقييم عام 4.8 مع رسم أشرطة التقييم
- 8 شهادات حقيقية بتفاصيل كاملة (جامعة / قطاع / تاريخ)
- فلترة بالدورة (المميزة / المكثفة / الشاملة / صريح ومتوازن)
- آراء سلبية وإيجابية لمزيد من المصداقية
- شارة "مشترك موثّق" لكل تقييم
- درجة STEP المحققة بارزة في كل بطاقة

### 🎨 التصميم والتجربة
- تصميم RTL كامل، خط Cairo
- متجاوب 100% (موبايل / تابلت / ديسكتوب)
- شريط إعلاني متحرك
- Hero مع خلفية وبادجات ثقة
- Trust Bar متحرك
- قسم مميزات، FAQ، About، Footer كامل
- لا أخطاء JavaScript
